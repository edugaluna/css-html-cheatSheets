CSS display reference

Basic syntax

display: <value>

Display block

They create a line break after the element and occupy the full width of their parent container.

.block-element {

display: block;

}

Display inline

They don’t create line breaks and occupy only the necessary width and height based on their content.

.inline-element {

display: inline;

}

Display inline-block

They behave like inline elements but allow you to set width, height, padding, and margins.

.inline-element {

display: inline-block;

}

Display none

They don’t take up any space in the document flow.

.inline-element {

display: none;

}

Display flex

Create a flex container, allowing you to build flexible and responsive layouts. Flex items inside the container can be positioned and resized using flexbox properties.

.inline-element {

display: flex;

}

Display grid

Create a grid container that enables you to define rows and columns, creating a two-dimensional grid system. Child elements inside the container can be placed in specific cells and adjusted for size and alignment.

.inline-element {

display: grid;

grid-template-columns: 1fr 1fr 1fr;

grid-gap: 10px;

}
